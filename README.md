# UWAL | Pong Game

> Classic Pong Game in [UWAL](https://github.com/UstymUkhman/uwal) and [WebGPU](https://developer.mozilla.org/en-US/docs/Web/API/WebGPU_API)

<!-- [![Pong Game]()]() -->

## Download

```bash
git clone https://github.com/UstymUkhman/uwal-pong-game.git
cd uwal-pong-game
```

## Develop

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```
